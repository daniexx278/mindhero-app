# Descripción del Proyecto

**My Hero** es una aplicación interactiva diseñada para adolescentes entre los **13 y 17 años**, enfocada en la educación emocional, la ansiedad y la depresión mediante experiencias gamificadas, ejercicios interactivos y herramientas de apoyo psicológico.

La aplicación utiliza una narrativa inmersiva estilo videojuego/apocalíptico donde el usuario se convierte en el “elegido” que debe aprender a enfrentar y comprender sus emociones para derrotar enemigos relacionados con la ansiedad y la depresión.

---

# 🎯 Objetivo del Proyecto

El objetivo principal de **My Hero** es:

- Enseñar habilidades emocionales y psicológicas de manera interactiva.
- Explicar conceptos de ansiedad y depresión mediante niveles gamificados.
- Ayudar al usuario a identificar pensamientos, emociones y conductas.
- Ofrecer herramientas rápidas de regulación emocional.
- Crear una experiencia visual moderna, inmersiva y amigable para adolescentes.
